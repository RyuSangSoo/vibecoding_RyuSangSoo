# KineLab Release Package

## 실행 환경

- 운영체제: Windows 10 / Windows 11 64-bit
- 배포 형식: Electron 기반 Windows 실행 파일
- 추가 런타임: 별도 설치 불필요

## 포함 파일

- `Release/KineLab-Portable-0.1.0.exe`
- `Release/KineLab-Setup-0.1.0.exe`

## 설치 절차

### Portable 버전

- 별도 설치 없이 exe 파일을 바로 실행합니다.

### Setup 버전

- 설치 마법사에 따라 설치 후 실행합니다.

## 실행 방법

### Portable 버전

1. `Release/KineLab-Portable-0.1.0.exe`를 더블클릭합니다.
2. 처음 실행 시 Windows 경고가 나오면 `추가 정보` -> `실행`을 선택합니다.

### Setup 버전

1. `Release/KineLab-Setup-0.1.0.exe`를 더블클릭합니다.
2. 설치 마법사를 완료합니다.
3. 설치 후 바탕화면 또는 시작 메뉴에서 KineLab을 실행합니다.

## 관리자 권한 필요 여부

- Portable 버전: 관리자 권한 불필요
- Setup 버전: 일반적으로 관리자 권한 불필요

## 참고

- 소스코드는 GitHub Repository의 `src/` 폴더에 포함되어 있습니다.
- 현재 버전은 1차 초안(MVP)입니다.
