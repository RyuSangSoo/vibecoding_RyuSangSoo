기획

어떤 문제를 해결하려 했는가

로봇공학 수업에서 DH 파라미터, 좌표계 변환, 정기구학은 핵심 개념이지만, 수식만으로는 링크 구조와 frame 관계를 직관적으로 이해하기 어렵다. 또한 교육 현장에서는 Linux 기반 대형 시뮬레이터보다, Windows PC에서 바로 실행되는 가벼운 도구가 더 실용적이다.

왜 이 문제를 선택했는가

실제 수업에서는 설치 환경보다 학습 흐름이 중요하다. 따라서 본 프로젝트는 범용 산업 시뮬레이터를 재현하는 대신, 교육에 필요한 기구학 중심 기능만 선별해 빠르게 실행되고 바로 이해되는 학습 도구를 만드는 데 집중했다.

구성물

프로그램 본체

React + Three.js로 작성된 메인 시뮬레이터 UI와 3D 매니퓰레이터 뷰를 포함한다.

수학/기구학 엔진

DH transform, 행렬 연산, Forward Kinematics 계산 로직으로 구성된다.

배포 패키지

Electron 기반으로 Windows용 Portable exe와 Setup exe를 함께 제공한다.

주요 기능 목록

  • 6축 매니퓰레이터 3D 시각화
  • Joint slider를 통한 자세 제어
  • DH Parameter Editor
  • End-Effector 위치/자세 표시
  • Transform Matrix (T06) 표시
  • Preset pose 지원
  • Axes / Housing 토글
  • Housing off 시 skeleton view
  • Axis legend로 X/Y/Z 색상 표시
  • Windows exe 실행 및 배포 지원

실현 방법

사용 기술

  • TypeScript: 계산 로직과 UI 상태를 안정적으로 관리
  • React: 제어 패널과 정보 UI 구성
  • Three.js: 3D 매니퓰레이터, local frame, housing 시각화
  • Vite: 빠른 개발 서버와 프론트엔드 빌드
  • Electron: Windows exe 패키징
  • Vitest: DH / Forward Kinematics 자동 테스트

결과

완성된 프로그램 설명

사용자는 Joint slider로 자세를 바꾸고, DH Parameter를 직접 수정하며 local frame과 End-Effector 결과가 어떻게 함께 변하는지 볼 수 있다. 교육용 도구답게 수식 개념과 시각 결과를 동시에 연결해 보여주는 구조를 목표로 했다.

자동 테스트 기준으로 DH transform, theta offset, Forward Kinematics 계산은 검증을 통과했다.

특장점

  • Windows 환경에서 바로 실행 가능한 교육용 시뮬레이터라는 점
  • 범용 물리 시뮬레이터보다 기구학 학습에 집중한 명확한 목적성
  • DH Parameter 수정과 3D 구조 변화가 즉시 연결되는 직관적 경험
  • local frame, skeleton, housing 등 학습 보조 시각화가 풍부한 점
  • 브라우저 버전과 exe 배포를 모두 가져갈 수 있는 확장성

한계점

  • 현재는 Forward Kinematics 중심이며 Inverse Kinematics는 아직 미구현 상태이다.
  • 동역학, 충돌, 물리 기반 상호작용은 포함하지 않았다.
  • 실제 산업용 로봇 모델과의 완전한 동일성을 목표로 하지는 않았다.
  • 코드 서명과 설치 메타데이터가 없어 배포 시 Windows 경고가 나올 수 있다.
  • 교육용 기록 저장, 과제 모드, 사용자 계정 기능은 아직 없다.
